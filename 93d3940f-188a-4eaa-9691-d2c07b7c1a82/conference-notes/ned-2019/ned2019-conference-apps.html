The Unsung Heroes of Console Apps: All the Features You’re Not Using, but Should

  • Console apps are helpful for those who work with multiple records at one time
  • Every time you open a new tab, it puts a lot of load on your browser
  • In a console, parent records open in different workspace tabs, and then subtabs for all the related records
  • Objects are listed on the left, related lists are in a split view
  • Tabs are retained if you close out and come back

Lightning Sales Console

  • Can add themes, change name of app, can change navigation style (until you save it)
  • Utility Bar – add Macros, Omni, Quick Text (see below)
  • Navigation Items – make sure users have access to whatever they need (accounts, contacts, opps, macros, quick text, omni supervisor
  • Navigation Rules – decide whether to open a workspace tab or subtab when you click on something
  • User profiles – grant access
  • If you’re editing a record, dialog is only open in that tab, can still work on other tabs
  • Keyboard shortcuts
  • Can choose to close all tabs when closing out
  • Can use list view as a workspace tab
  • Lightning Extension – adds icon to browser (Chrome for now) and console. When a user clicks on it, can enable component customization (show or hide components), dark mode, link grabber (when you copy, paste, or send a link, will add it as another tab in the console)

Setting Up Lightning Sales Console

  • Go to Lightning App Builder > Create New Page > Record Page
  • Label: Pinned Account, Object: Account
  • Pinned Regions – Drag over highlights panel into the layout. Regardless of what record you’re in, the account highlights panel will always be displayed.
  • Activate for sales console in app default

Edit/Customize Tabs

  • Can set subtabs as workspace tabs
  • Customize tabs

Components to Use with Lightning Sales Console

Omni Channel

Rather than workers picking records from a queue, supervisors can push work to users and monitor their capacity and activity

Macros

  • Great for turning repetitive, quick actions into one click. Like reports: you can save them in a public folder but default is private.
  • Setup > Macro Settings
  • Can use logic in your macros

Quick Text

  • Great for sending template emails. Default private also.
  • Insert merge fields into text template
  • When you’re in an email/activity, find and click on the quick text bubble and search for the quick text you want to use
  • Different from an email template in that it’s just a snippet of text that you use frequently but would build upon, like an introductory line.
