Program Partner and Classroom Management in Salesforce

Like sites, every program partner has an Account record in Salesforce, with Organization Agreement records linked to it for each program partner agreement term. Having an Organization Agreement for the current year marks the program partner Account as Active, which makes it appear in JStats.

Classroom records are the link between Children/Classroom Registrations, Member Teams, and site and program partner Accounts. Each classroom should have one record that gets re-used year over year (this is different from SAM, where a new record would be created each year). Like program partner Accounts, Classrooms need to be marked as Active in order to appear in JStats.

To ensure that a particular program partner has up-to-date contact, agreement, and classroom information:

  1. Go to the program partner's Account record in Salesforce and click the Setup Organization Agreement button on the right-hand side.
    • If there is no existing Account record, go to your Home page and click the Setup Program Partner Organization Agreement button on the right-hand side.
  2. Follow the guided instructions to verify the program partner's contact information and update or create an Organization Agreement record for the current/upcoming year(s). This will check the Active box on the program partner's Account record, which makes it appear in JStats.   3. From the Account or the current year's Organization Agreement, click the Send Program Partner Agreement button to send the Primary Contact the full Program Partner Agreement form, or click the Send Program Partner Classroom Update button to send them the yearly Program Partner Classrooms update form. These forms will prefill with information that's already in Salesforce, which they can update as needed.
  4. When the Primary Contact submits the Program Partner Agreement form, it will redirect them to the Program Partner Classrooms form (or they can use the link you sent). The Agreement form will automatically save as a Secure Document linked to the Organization Agreement, and the Classrooms form will create/update the Classroom records linked to the Account, which will make them appear in JStats.

Paper Forms/Manual Updates

If your program partner has filled out paper forms or signed pdf versions, you still need to complete steps 1 and 2 in order to maintain updated information and make the Account and Classrooms appear in JStats, but you can ignore steps 3 and 4. If you want to uploading pdfs of the paper forms to Salesforce, you can use the Add Secure Document tool on the Organization Agreement. You should also update the Classrooms manually (avoid doing this if you've already sent the Program Partner Agreement form/Classroom Update form electronically and you're just waiting for them to submit it, because that could lead to duplicate records):

  • Click on the Classrooms (Site) related list on the site's Account record (or the Classrooms (Program Partner) related list on the program partner's Account record) to see if there is already a Classroom record for that classroom (see image 1 below)
    • If there is an existing record, even if it's from a previous year, use that one instead of creating a new one.
      • Make sure the JSD Active box is checked so that it appears in JStats.
      • Update the Primary Teacher, if needed.
    • If there is no existing record:
      • Click New (image 2)
      • Enter the Classroom Name
      • Site: Search and select with the site's Account record (will autofill if you're creating the Classroom from the site)
      • Program Partner: Search and select the program partner's Account record (will autofill if you're creating the Classroom from the program partner)
      • Primary Teacher: Search and select the teacher's Contact record
      • JSD Active: Checking this box will make the Classroom appear in JStats, and will include that Classroom in the site's and program partner's total classrooms number. If a previous year's Classroom is no longer being used, uncheck this box.
      • Non-traditional Classroom: used by Eval, leave blank
      • Comparison Classroom: used by Eval, leave blank
      • Click Save (image 3)

For both program partner Accounts and Classrooms, you can uncheck the Active box to remove them from your list in JStats. This step is not automated yet.
