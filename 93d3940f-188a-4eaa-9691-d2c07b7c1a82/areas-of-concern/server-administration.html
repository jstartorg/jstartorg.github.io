Description

Jumpstart runs servers both On-Premise and in the Cloud. The on-prem servers are chiefly in Boston, but some offices (like Los Angeles) will have a single older model laptop deployed as a Hyper-V host to run edge resources like a Domain Controller. The servers in the cloud are chiefly located in AWS, but there is one server in Azure currently. They all form a network over VPN connections, on-prem and AWS through Meraki and Azure through a third-party connection to the Meraki device in AWS.
