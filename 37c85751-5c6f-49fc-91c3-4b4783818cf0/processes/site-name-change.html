Description

Changing the name of a site is very involved and not to be done lightly. It is also somewhat time sensitive as systems sincing

Salesforce

  • Update the Active Directory Site Code to something appropriate
  • Set the MyJStart ID to the name of the site using only lowercase alphanumeric characters and hyphens instead of spaces (eg St. Mary's College becomes st-marys-college)
  • Update the Account Name
  • Update the Sharing Rule for the site
  • Update the Public Group to the name of the site using Title Case alphanumeric characters and underscores instead of spaces (eg St. Mary's College becomes St_Marys_College)
    • if the resulting site name is greater than 40 characters it will cut off the end, but that should be fine as it will match on the start of the Active Directory group name
